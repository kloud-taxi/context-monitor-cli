# コマンドの一覧

## `npm i -g @kloud-taxi/cmc`

CLIのインストール

## `cmc`

- アプリケーションを起動します。タスクを入力するガイドを表示します。
- 初回だけAPIキーを入力するためのガイドが表示されます。
- APIキーは`~/.cmc/config.json`に保存されます。
- アプリケーションの監視を終了するには、`ctrl + C`を入力します。

### `cmc "タスクの概要"`

- タスクを入力するガイドを省略できます。

## `cmc -v`

バージョンの表示

## `cmc -h`

ヘルプの表示

## `cmc verify-log /path/to/file`

ログの検証
